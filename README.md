# 🎮 Rock-Paper-Scissors Game

A simple **C++ console-based Rock-Paper-Scissors game** where you play against the computer.  
The computer’s move is generated using a random number generator, ensuring fair gameplay each round.

---

# 🚀 Features
- Play multiple rounds until you choose to exit.
- Randomized computer moves using `std::mt19937` (Mersenne Twister engine).
- Clear instructions and results displayed after each round.
- Lightweight and beginner-friendly C++ implementation.

---

# 🕹️ How to Play
1. Run the program.
2. Enter your move:
   - `1` → Rock  
   - `2` → Paper  
   - `3` → Scissors  
3. The computer will randomly choose its move.
4. The result (Win/Lose/Tie) is displayed.
5. Continue playing or exit when prompted.
